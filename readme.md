# Fresh-Mart

Fresh-Mart is a frontend grocery store website built using plain HTML, CSS, and JavaScript.  
The project focuses on creating a clean shopping experience with product listings, a cart layout, and a checkout flow without using external frameworks.

This repository is intended for learning, practice, and showcasing frontend development skills.

---

## Overview

Fresh-Mart simulates an online supermarket where users can browse products, view product details, add items to a cart, and proceed to checkout.  
The interface is simple, responsive, and designed to work well on both desktop and mobile devices.

---

## Features

- Responsive navigation bar with cart counter  
- Hero section for promotions  
- Categories section  
- Product cards with prices and actions  
- Product details page layout  
- Cart page with quantity controls  
- Checkout form UI  
- Footer with branding and links
